# Meu Portfólio Pessoal

Este é o projeto do meu site de portfólio, desenvolvido utilizando HTML, CSS e JavaScript.

## 🚀 Tecnologias Utilizadas
- **HTML5:** Utilizando tags semânticas apropriadas (header, nav, main, section, footer).
- **CSS3:** Layout totalmente estruturado com **Flexbox**.
- **JavaScript:** Implementação de recurso de alternância de Tema Claro/Escuro.

## 📋 Critérios Atendidos
- Layout estruturado puramente com Flexbox.
- Diversidade de componentes e tags semânticas.
- Navegação simples com três páginas (`index.html`, `projetos.html`, `contato.html`).
- Recurso em JavaScript incluído.
- Marcação validada (W3C Markup Validation).

## 🔧 Como visualizar
Basta fazer o download deste repositório e abrir o arquivo `index.html` em qualquer navegador de sua preferência.
